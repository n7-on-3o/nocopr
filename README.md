# üèÆ Niri: Catppuccin Macchiato Rice
> A soft, "roundy" Wayland environment with a Japanese minimalist aesthetic.

This repository contains my personal dotfiles for a **Niri**-based workflow. The theme is built around the **Catppuccin Macchiato** palette, focusing on high corner radii, generous gaps, and Japanese typography.
It has been a challenge to see how I can build a complete system from a base Fedora install (Fedora Everything) and using no copr packages
